***Colab Admin PHP***
**Must be running Service**
```
httpd or nginx on port 80
pgsql on port 5432
socket server on port 7000
```
**CLI List**

Server related setup
```
$ addon/setup-minimal-server.sh   # run setup install required software on ubuntu 20.04
$ addon/setup-tts.sh # install coqui-tts python3.8 module, and indonesia tts support
$ addon/setup-pgsql # install & setup postgres
```

Database related cli
```
$ addon/restore-db.sh   # restore db and load json dumps
$ addon/backup-db.sh    # backup db and save to json dumps
$ addon/fixtures-db.sh  # load fixtures db sample rows

```

**Socket Api List**
Api for create Background process job
Socket Io Server and Http Server running on port 7000
```
[POST]http://localhost:7000/job?uuid=caf6a3f0-c501-47b0-baf9-cf305dbd27f4&job_name=tts

params
uuid=machine-autogenerated-uuidv4  # currently for socket identification
job_name=           # job name currently defined in job-routes.js

request body
x-www-form-urlencoded
text=               # for example for job tts require text param as form data

example output 

{
    "status": true,
    "message": "Job tts created with id 1"
}
```
